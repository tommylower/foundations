# design files

this directory contains .pen files for pencil.dev.

## setup
1. install the pencil.dev extension for VS Code or Cursor
2. authenticate with Claude Code CLI
3. open any .pen file to edit visually

## how it works
- .pen files are version-controlled design assets
- pencil reads tailwind tokens from tailwind.config.ts automatically
- visual edits write back to components
- naming: [section-name].pen (e.g. hero.pen, services.pen)
