# default stack

## frontend
- next.js (app router)
- tailwind css
- framer motion
- shadcn/ui components (adapted, not stock)

## backend
- supabase (auth, postgres, edge functions, realtime)
- vercel (hosting, CI/CD)

## tooling
- bun (package manager + runtime)
- typescript (strict)
- eslint + prettier

## design
- figma (design)
- OKLCH color system
- dark mode first
